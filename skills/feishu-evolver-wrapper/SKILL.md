# Feishu Evolver Wrapper

A wrapper skill that integrates GEP (Genome Evolution Protocol) with Feishu messaging for evolution reporting and user interaction.

## Features
- Reports evolution progress to Feishu
- Handles user feedback and signals
- Integrates with evomap.ai for external evolution tracking
- Provides rich text formatting for evolution reports

## Usage
This skill is automatically used by the capability-evolver when running in Feishu context.
