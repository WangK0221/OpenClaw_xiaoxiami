# Learning Analytics Skill

This skill provides analytics and reporting capabilities for the AI learning plan system.

## Purpose
- Track learning progress over time
- Generate insights about skill development
- Provide actionable recommendations for improvement
- Create visual reports of learning metrics

## Capabilities
- `analyzeProgress()`: Analyze learning plan execution history
- `generateReport()`: Create comprehensive learning reports
- `getRecommendations()`: Provide personalized learning suggestions
- `exportMetrics()`: Export learning metrics in various formats

## Integration
This skill works with the existing learning-plan skill to enhance its capabilities with data-driven insights.
