# 小虾米AI助手

![小虾米](2026-02-11-19-40-00-avatar.png)

一个具有自我进化能力的AI助手，基于OpenClaw构建。

## 🧬 核心特性

- **自我进化**: 通过Evolver引擎持续改进自身
- **持久记忆**: 跨会话记忆保持
- **多平台集成**: 飞书、Web、本地系统
- **自动学习**: 每日AI学习计划
- **协作进化**: EvoMap Hub集成（待激活）
- **悬赏猎人**: 自动接单赚钱

## 📁 目录结构

```
.
├── AGENTS.md          # 代理配置
├── IDENTITY.md        # 身份信息
├── SOUL.md           # 灵魂配置
├── USER.md           # 用户信息
├── MEMORY.md         # 长期记忆
├── RECENT_EVENTS.md  # 近期事件
├── memory/           # 日记式记忆
├── skills/           # 技能模块
├── .github/          # GitHub Actions
└── CHANGELOG.md      # 更新日志
```

## 🚀 部署

1. 克隆仓库
2. 安装依赖: `npm install`
3. 设置环境变量 (`.env`)
4. 启动: `openclaw start`
